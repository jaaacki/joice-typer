# JoiceTyper logo assets

Style: B4 — accent border, light fill. Rounded-square container with a
1.5px terracotta stroke, warm off-white fill, ink waveform bars, and an
accent J hook.

## Files
- joicetyper-logo.svg — default (256 intrinsic, scales)
- joicetyper-logo-{16,32,64,128,256,512}.svg — intrinsic-size SVGs
- joicetyper-logo-dark.svg — dark surface, white bars
- joicetyper-logo-mono.svg — single-color (print / favicon)
- joicetyper-logo-themeable.svg — drives bars from currentColor;
  accent + fill from --logo-accent and --logo-fill CSS vars
- joicetyper-logo-{16..1024}.png — raster
- joicetyper-logo-dark-512.png — dark raster

## Colors
- Ink     #382d25
- Accent  #c15a3b
- Surface #fefcf9
- Dark    #1a1612

## Usage
```html
<img src="assets/logo/joicetyper-logo.svg" alt="JoiceTyper" width="48" height="48">
```
