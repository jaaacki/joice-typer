# JoiceTyper logo — B3 (dark squircle · border)

Dark squircle (#1a1612) with a subtle 1px outline (#2a2320), white waveform bars,
and accent J hook (#c15a3b). Works on both light and dark surfaces.

## Files
- joicetyper-logo.svg — default (256)
- joicetyper-logo-{16,32,64,128,256,512}.svg — intrinsic-size
- joicetyper-logo-themeable.svg — currentColor bars; --logo-accent, --logo-fill, --logo-border CSS vars
- joicetyper-logo-{16..1024}.png — raster

## Colors
- Dark fill    #1a1612
- Dark border  #2a2320
- Bars         #ffffff
- Accent       #c15a3b
